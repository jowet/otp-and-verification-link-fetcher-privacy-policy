Privacy Policy for OTP Extractor

Last updated: December 2024

Overview

OTP Extractor is a Chrome extension that helps users extract OTP codes and verification links from Gmail messages. This privacy policy explains how we handle your data.

Important: We do not collect, store, or transmit any personal data to external servers. All data processing happens locally on your device.

Data Collection

Our extension collects the following data:

  • Gmail Messages: We read your Gmail messages to extract OTP codes and verification links
  • Authentication Tokens: Temporary Gmail access tokens for API authentication
  • Extracted Data: OTP codes and verification links found in your emails

Data Storage

All data is stored locally on your device using Chrome's storage API:

  • No data is sent to external servers
  • No data is shared with third parties
  • You can clear all data anytime through Chrome's extension settings

Data Usage

We use your data to:

  • Extract OTP codes from incoming emails
  • Find verification links in your messages
  • Display extracted codes and links for easy access
  • Maintain your login session with Gmail

Permissions

Our extension requires these permissions:

  • Identity: For secure Gmail authentication using OAuth 2.0
  • Storage: To store OTP codes and verification links locally
  • ActiveTab: To display notifications when new OTP emails arrive

Data Security

We implement the following security measures:

  • Uses Google's secure OAuth 2.0 authentication
  • No passwords or credentials are stored
  • All data is encrypted using Chrome's built-in security
  • Access tokens expire automatically

Your Rights

You have the right to:

  • Clear all stored data through Chrome's extension settings
  • Revoke Gmail access through your Google Account settings
  • Uninstall the extension at any time
  • Contact us with privacy concerns

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the "Last updated" date above.
